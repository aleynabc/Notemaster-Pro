# NoteMaster Pro 📝

A React Native-based smart order management and note-taking application featuring a modern UI designed via Google AI Studio, with integrated local notification support.

## 🚀 Features
* **Smart Order Tracking:** Manage your orders and customers efficiently.
* **Local Notifications:** Get notified before your order deadlines.
* **Modern UI:** Clean and user-friendly interface designed with AI assistance.
* **Android Support:** Fully compatible with Android devices via Capacitor.

## 🛠 Tech Stack
* **Framework:** React / Vite
* **Mobile Bridge:** Capacitor
* **UI Design:** Google AI Studio / Tailwind CSS
* **Notifications:** Capacitor Local Notifications
